# データの見方

  <Box sx={{ ml: 2, my: 3 }}>
    ## バトルランク
    テーブルデータのキーに相当します。<br/>
    パーティ加入時ステータス、せんせいの修行におけるステータス成長上限の算出では現在バトルランク（1～200）が参照されます。<br/>
    それ以外の目的では敵バトルランク（1～250）が参照されます。

    バトルランクの詳細については [ロジック ＞ バトルランク](/saga-eb/logic/battle-ranks) を参照してください。

    ## 武器威力
    敵の武器威力です。<br/>
    唯一の例外を除き、ほぼ全ての敵キャラクターにとって味方の武器威力と同等の役割を果たします。

    ## スキルレベル
    敵のスキルレベルです。<br/>
    味方のスキルレベルと同等の役割を果たします。

    ## 基礎属性耐性
    敵の各種属性耐性の基礎となる内部値です。<br/>
    敵種族に設定されている属性耐性（種族属性耐性）、敵キャラクターに設定されている属性耐性（固有属性耐性）との組み合わせによって最終的な属性耐性が決定されます。

    ## 基礎HP
    敵の最大HPの基礎となる内部値です。<br/>
    敵キャラクターに設定されている最大HP（固有最大HP）との組み合わせによって最終的な最大HPが決定されます。

    ## 基礎能力値
    敵の各種能力値の基礎となる内部値です。<br/>
    敵キャラクターに設定されている能力値（固有能力値）との組み合わせによって最終的な能力値が決定されます。

    ## 成長ランク
    パーティ加入時ステータス、ステータス成長上限に影響する内部値です。<br/>
    バトル勝利におけるステータス成長上限の算出では味方キャラクターに設定されている成長適正との組み合わせによって最終的な成長上限が決定されます。
